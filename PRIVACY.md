# Privacy Policy — Aurora Personal Hub

Last updated: June 12, 2026

## What this is

Aurora is a personal portfolio project by Mersaid Zhaxybaev for 
aggregating my own athletic data. It is not a commercial product.

## Data collected via Whoop

Aurora requests the following data via Whoop OAuth:
- Recovery metrics (HRV, RHR, recovery score)
- Sleep metrics (duration, stages)
- Workout activities
- Cycle metrics (strain, average HR)
- Profile (name, email)
- Body measurements (height, weight, max HR)

## How data is used

- Stored in a private PostgreSQL database, accessible only to the project owner
- Used solely for personal training analysis and portfolio demonstration
- Never shared with third parties
- Never sold or monetized

## Data retention

User may revoke access at any time via Whoop's app management page.
Upon revocation, all stored data is deleted within 30 days.
